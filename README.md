# Expense Tracker

A full-stack expense tracking application with AI-powered chatbot assistance.

## Features

- AI Chat Assistant (Google Gemini)
- User Authentication (JWT)
- Expense & Budget Management
- Category-based Organization
- Filtering & Search
- Feature Flag Service

## Tech Stack

**Frontend:** React, CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**AI:** Google Gemini API  
**Feature Flags:** Flask, SQLAlchemy  
**Auth:** JWT, bcrypt
